##  Pre-Entrega

Sitio web de wallpapers basados en naturaleza (son gratuitos). El objetivo es presentar una galería simple, responsiva y visualmente cuidada; que permita visualizar y "descargar" imágenes de paisajes naturales.

## Características:

- Estructura HTML semántica.
- Estilos aplicados (grid, flexbox, media queries).
- Tipografía mediante Google Fonts.
- Imágenes con efecto hover.
- Formulario de contacto funcional (vía Formspree).
- Diseño adaptable (responsive).
- Despliegue vía GitHub Pages.
